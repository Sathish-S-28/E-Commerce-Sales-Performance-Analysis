# 🛒 E-Commerce Sales Performance Analysis

## 📌 Project Overview
This project analyzes e-commerce sales data to uncover insights related to customer behavior, product performance, pricing, and delivery efficiency.

---

## 🎯 Objectives
- Customer segmentation and behavior analysis  
- Product and pricing optimization  
- Delivery performance analysis  

---

## 🛠️ Tools & Technologies
- Python (Pandas, NumPy, Matplotlib, Seaborn, Scikit-learn)
- SQL
- Power BI
- Statistics and Machine Learning

---

## 📊 Key Insights
- Furniture category generates highest revenue  
- UPI is the most used payment method  
- Customers show high satisfaction (4–5 ratings)  
- Sales show seasonal growth trends  

---

## 🤖 Machine Learning Models
- Linear Regression (R² Score: 0.90)  
- KNN Classification (F1 Score: 0.86)  
- K-Means Clustering (Silhouette Score: 0.90)
